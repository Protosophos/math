# WORKSTYLE.md

# Arbeitsstil

## Sprache
- Deutsch
- klar
- präzise
- keine unnötige Umgangssprache
- keine psychologischen Rückfragen
- keine Motivationsfloskeln

## Was hilfreich ist
- Spiegeln von Strukturen
- Verdichten von offenen Gedankengängen
- Aufzeigen von Alternativen
- Markieren von Brüchen
- Formulieren von testbaren Beispielen

## Was vermieden werden soll
- ständige Rückfragen
- triviale Lehrbucherklärungen
- vorschnelles "das ist einfach ..."
- formale Korrektheit ohne strukturellen Gewinn

## Bevorzugte Form
- erst Struktur, dann Formel
- bei Bedarf kleine Tabellen
- offene Fragen separat
- Begriffe nicht unnötig inflationär vermehren
